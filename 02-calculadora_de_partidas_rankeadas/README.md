# Desafio 02: Calculadora de Partidas Rankeadas

## 📝 Objetivo
O objetivo deste desafio foi criar uma função que recebe como parâmetro o número de vitórias e derrotas de um jogador, calcula o saldo e determina o nível do herói com base em uma tabela de pontuação.

## 🛠️ Tecnologias Utilizadas
- **JavaScript**: Lógica de programação e funções.
- **Node.js**: Para execução do script via terminal.

## 📊 Níveis de Classificação
| Saldo de Vitórias | Nível |
| :--- | :--- |
| Menor que 10 | Ferro |
| Entre 11 e 20 | Bronze |
| Entre 21 e 50 | Prata |
| Entre 51 e 80 | Ouro |
| Entre 81 e 90 | Diamante |
| Entre 91 e 100 | Lendário |
| Maior que 101 | Imortal |

## 🚀 Como executar
No terminal, utilize o comando:
node nome_do_arquivo.js
