# 🦸‍♂️ Classificador de Nível de Herói

Projeto desenvolvido como desafio de lógica de programação.

---

## 📌 Sobre o Projeto

Este programa classifica o nível de um herói com base na quantidade de XP informada.

## 🏆 Níveis

- Ferro
- Bronze
- Prata
- Ouro
- Platina
- Ascendente
- Imortal
- Radiante

---

## 💻 Tecnologias utilizadas

- JavaScript
- Node.js

---

## 🚀 Como executar

```bash
node index.js
```
